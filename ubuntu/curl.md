### Normal request

```sh
curl https://sharklabs.com.br/
```

### Send HTTP HEAD request

```sh
curl -I https://sharklabs.com.br/
```
Important link: https://developer.mozilla.org/pt-BR/docs/Web/HTTP/Methods/HEAD


### Send any request and show the RESPONSE HEADERS

```sh
curl -i https://sharklabs.com.br/
```
