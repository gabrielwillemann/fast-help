### DNS: Advanced Zone Editor

- Type A
  - Goal: Redirect for subdomains
- Type AAAA
  - Equal type A, but for IPV6
- Type CNAME
  - It's a alias
  - Recommend to redirect for other domain
- Type TXT
  - Free entry
  - You can write anything
